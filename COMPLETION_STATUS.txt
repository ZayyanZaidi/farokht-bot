🚀 SUPPORTED QUERY TYPES

1. Brand-Specific Search
   Example: "I'm looking for kurta from Khaadi"
   → Returns: Khaadi kurta products

2. Price-Filtered Recommendations
   Example: "Can you recommend a shoe under Rs. 2000?"
   → Returns: Products matching price filter

3. Category Search
   Example: "Show me bags"
   → Returns: All bags in database

4. Color Filtering
   Example: "Red kurtas please"
   → Returns: Red-colored kurtas

5. Trending/Best-Sellers
   Example: "What's trending right now?"
   → Returns: Popular/high-priced items

6. Similar Products
   Example: "Show me similar products to kurta"
   → Returns: Related items

7. Budget Alternatives
   Example: "Something cheaper than Rs. 3000"
   → Returns: Products within budget

8. Multi-Language Support
   English: "Show me kurtas"
   Urdu: "کرتے دکھاؤ"
   Roman Urdu: "Kurty dikhao"

  - Deployment: Google Cloud Run

  - Contextual filtering

  - Sync method: API polling + manual trigger

🚀 DEPLOYMENT READY

To deploy to production (GCloud):

1. Ensure all changes are committed:
   git add -A
   git commit -m "Enhanced query handling and improved backend"

2. Deploy to GCloud:
   gcloud run deploy farokht-bot-backend \
     --source . \
     --region us-central1 \
     --allow-unauthenticated

3. Verify deployment:
   curl https://farokht-bot-backend-784756226072.us-central1.run.app/health
